## Usage
Type a `?` in the presentation to open a dialog listing the available shortcuts.

### Setup
```bash
$ npm install
```

### Start presentation
```bash
$ npm start
```

### Build static website
```bash
$ npm run build
```

### Release static website and PDF export to GitHub Pages
```bash
$ npm run release "Release v1.0.0"
```

### Export as PDF
```bash
$ npm run pdf
```

### Development
```bash
$ npm run dev
```
