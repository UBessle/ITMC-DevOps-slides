# techcamp-devops-slides
Slides for the DevOps keynote at TechCamp 2019 https://techcamp.hamburg/

## Setup
```bash
$ npm install
```

## Start presentation
```bash
$ npm start
```

## Build static website
```bash
$ npm run build
```

## Export as PDF
```bash
$ npm run pdf
```

## Development
```bash
$ npm run dev
```
